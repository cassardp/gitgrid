Privacy Policy

Last updated: April 13, 2026

What GitGrid is

GitGrid is a portfolio generator that turns your GitHub repositories into a public profile page. It is operated by Patrice Cassard as a personal project.

Data we collect

When you log in with GitHub, we store:

  • GitHub profile — username, display name, bio, avatar URL, blog, Twitter handle
  • Repository metadata — name, description, language, stars, visibility, homepage URL. No source code is accessed.
  • Access token — encrypted at rest (AES-256-GCM), used only to sync your repos from the GitHub API
  • Configuration — layout preferences, card order, hidden repos, uploaded images

Images

Screenshots you upload are stored on Cloudflare R2 and served publicly at your portfolio URL. Images are processed client-side (resized, converted to WebP) before upload.

Cookies

We use a single session cookie (gitgrid_session) to keep you logged in. It is HttpOnly, Secure, and SameSite=Lax. No tracking cookies, no analytics, no third-party scripts.

Third-party services

  • GitHub — OAuth authentication and repository data (GitHub Privacy Statement)
  • Cloudflare — hosting, database, image storage (Cloudflare Privacy Policy)
  • Google Fonts — font loading (Google Privacy Policy)

Data retention

Your data is kept as long as your account exists. You can delete your account at any time from the Settings menu — this permanently removes your profile, configuration, access token, and all uploaded images.

Your portfolio is public

Your portfolio page is accessible to anyone at gitgrid.app/username. Public repo metadata and any uploaded screenshots are visible to visitors. Private repos without a homepage URL are automatically hidden from visitors.
